## Deployment

To deploy this Jupyter Notebook run

1. Abre tu terminal y navega al directorio del proyecto.
2. Ejecuta Jupyter Notebook:

   ```
   jupyter notebook
   ```
3. Abre el archivo `.ipynb` en la interfaz de Jupyter que se abre en tu navegador.
